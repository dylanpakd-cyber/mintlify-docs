Every iPhone needs the same one-time setup, and it covers **every** platform: set a device up once and it can warm and post on TikTok, Instagram, X, and LinkedIn.

- **Developer Mode** on (Settings → Privacy & Security → Developer Mode).
- **UI Automation** on (Settings → Developer → Enable UI Automation): required, or no automation runs.
- **Appearance = Light** and **StandBy = Off**: the two settings that cause most failed sessions.
- **Auto-Lock = Never** and **Brightness ≥ 50%**.
- The **Warmr runner** installed on the device.

See [Prepare each iPhone](/setup/iphone-settings) for the full checklist, then [Install the runner](/setup/install-runner).
